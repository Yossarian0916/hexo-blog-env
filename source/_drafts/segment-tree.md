# Segment Tree

线段树呢，首先是树状的数据结构，每个节点代表一个区间或线段。

以二叉树的形式来储存一区间上的某一特性（像区间和，区间里的最小值等）。可以用指针方式实现二叉树，亦可以用树状数组。线段树的操作都基于递归和回溯。
